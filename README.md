## Run Locally

Clone the project

```bash
  git clone https://github.com/hugorince/How-Many-Capitals-Next.git
```

Go to the project directory

```bash
  cd How-Many-Capitals-Next
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
